# cHttpClient 概述

## 简介

`cHttpClient` 是一个基于 Microsoft WinHTTP Services 5.1 组件封装的 HTTP 客户端类，专为 TwinBASIC/VBA 环境设计。它提供了现代化的链式调用 API，简化了 HTTP 请求的处理流程。

## 设计目标

1. **简洁易用** - 通过链式调用减少代码量
2. **功能完整** - 覆盖常见的 HTTP 请求场景
3. **灵活可控** - 支持自定义请求头、Cookie、超时等参数
4. **错误友好** - 清晰的状态码处理和错误信息

## 核心功能

| 功能 | 说明 |
|------|------|
| HTTP 方法 | GET、POST、PUT、DELETE、OPTIONS |
| 同步/异步 | 支持同步等待和异步回调 |
| 数据格式 | JSON、表单、纯文本、二进制 |
| 重定向处理 | 自动/手动跟随 3xx 重定向 |
| 编码处理 | 自动 UTF-8 编解码 |
| SSL/TLS | 自动处理，支持忽略证书错误 |
| 调试日志 | 完整的请求/响应信息记录 |

## 适用场景

- REST API 调用
- Web 服务集成
- 数据抓取
- 文件上传下载
- OAuth 认证流程
